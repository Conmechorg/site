Consciousness Mechanics (CM)

Consciousness Mechanics (CM) is a structured research framework. It develops and applies the Consciousness Component Model (CCM) — a formal ladder of control capabilities (CL1…CLn) expressed as explicit mechanism.

A biology-first research framework modelling consciousness as control architecture.

The Consciousness Component Model (CCM) introduces capabilities only when required by evidence — each defined as bounded mechanism (CL1…CLn).

Overview

CM is the public home of an in-progress research framework — built to be explicit, conservative, and mechanistically checkable. It is not a finished theory, and not a position statement.

What this site will publish
  • Component definitions that behave like engineering specs (state, gates, update rules).
  • Diagrams and pseudocode that show how control capability scales across organisms.
  • Anchors and discriminators: what would count as evidence for / against a capability.
What it will avoid
  • Metaphor-led explanations that can’t be operationalised.
  • Assuming higher cognition by default and projecting it downward.
  • Over-claiming beyond the defined scope of each component.

If you’re evaluating whether CCM has substance: the fastest indicator is not prose — it’s whether each component can be read as a bounded mechanism with a clear ceiling, and whether the ladder remains coherent as it climbs.

Approach

The framework is developed under constraints designed to prevent conceptual drift and over-claim. These are “rules of the build”, not rhetoric.

Grounding & mechanism
  • Biology-first: define capabilities from whole-organism control problems.
  • Operational spec: components are expressed as variables, gates, and update rules.
  • No narrative substitution: explanation must cash out in mechanism.
Discipline & scope
  • Incremental ladder: add capability only when lower levels cannot explain the behaviour.
  • Conservative ceilings: absence of a component is a meaningful claim.
  • Scope boundaries: every component states what it does not claim.

The goal is a framework that stays falsifiable, extensible, and comparable across life — without importing human-specific assumptions as default explanations.
